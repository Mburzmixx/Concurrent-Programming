# Concurrent-Programming
This repository stores projects done during course "Concurrent Programming".

## Factory
**My code can be found in folder**: `Factory/solv/plant`.

**The main code** is located in `Factory/solv/plant/solution.c`.

`Factory/solv/plant/linkedList.c` contains implementation of a **linked list** which is declared in `Factory/solv/plant/linkedList.h`.

In order to build run:

`cd ./Factory`

`mkdir build`

`cd ./build`

`cmake ..`

`make`

Test run can be done using:

`./demo/demo`
